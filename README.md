# Cyber Help

## projet description

Application proposée pour résoudre ces situations délicates avec un système d’alerte déclenchées par l’élève, informant un médiateur qu’il y a un problème dans sa classe.

## Stack

- nodejs
- express
- mongodb
- jade
- ionic
